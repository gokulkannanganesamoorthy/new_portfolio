# Personal Portfolio - "New World" Edition

A high-performance, immersive portfolio website built with React, Vite, and Tailwind CSS. Features brutallist aesthetics, smooth scrolling (Lenis), and advanced animations (Framer Motion).

## Tech Stack

- **Framework**: React 19 + Vite
- **Styling**: Tailwind CSS v3 + Custom CSS Variables
- **Animation**: Framer Motion
- **Scroll**: @studio-freight/react-lenis
- **Icons**: Lucide React

## Getting Started

1. **Install Dependencies**

   ```bash
   npm install
   ```

2. **Run Development Server**

   ```bash
   npm run dev
   ```

3. **Build for Production**
   ```bash
   npm run build
   ```

## Deployment

This project is optimized for deployment on Vercel or Netlify.

- **Build Command**: `npm run build`
- **Output Directory**: `dist`

## Features

- **Immersive Hero**: Parallax scroll effects.
- **Horizontal Portfolio**: "Deck" style project showcase.
- **Magnetic Interactions**: Buttons and cursor physics.
- **Floating Navigation**: "Island" style navbar.
- **Noise Texture**: Custom grain overlay.
